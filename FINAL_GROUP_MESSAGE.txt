## 🧠 HOW IT WORKS:

### RECOMMENDATION ALGORITHM (Smart):
Scoring combines:
- Rating (40% weight) - Higher rated = better
- Price Value (30% weight) - Lower price = better
- Popularity (20% weight) - More reviews = better
- Tag Matching (10% weight) - Your preferences = better

Example: You search "Electronics under $500, wireless"
→ System finds all wireless electronics under $500
→ Scores each by rating, price value, reviews, tags
→ Shows best matches first!

### SEARCH ALGORITHM:
Matches product name (priority 3)
Matches brand (priority 2)
Matches tags/features (priority 1)
Sorts by priority + rating

### SIMILAR PRODUCTS:
Finds products in same category
Similar price range (±30%)
Returns highest-rated options

## 💻 QUICK CODE SNIPPETS:

### To add a product (for testing):
```python
from recommender_v2 import ShoppingAssistant

assistant = ShoppingAssistant('products.json')

# Get recommendations
recommendations = assistant.smart_recommend(
    category='Electronics',
    max_price=500,
    min_rating=4.0,
    tags=['wireless'],
    limit=5
)

for r in recommendations:
    print(f"{r['name']} - ${r['price']} - Score: {r['recommendation_score']:.1f}")
```

### To search:
```python
results = assistant.search_products('apple', limit=5)
for r in results:
    print(f"{r['name']} - {r['brand']}")
```

### To find similar products:
```python
similar = assistant.find_similar(product_id=1, limit=3)
```

### To get category stats:
```python
stats = assistant.get_category_stats('Electronics')
print(stats)  # Shows avg price, avg rating, brands, etc.
```
